## 项目简介
这是一个基于 React 的 Web 应用程序， 对接了 Dify API。

## 已完成功能
- dify 应用配置
- chat api 对接
- 文件上传对接
- 引用文档展示
- echarts 图表输出

## 技术栈
- 前端：React

## 开发环境要求
- Node.js 20+
- npm 或 yarn

## 安装说明

### 前端设置
1. 安装依赖：
```bash
npm install
# 或
yarn install
```

## 运行项目

### 开发环境

2. 启动前端开发服务器：
```bash
npm run start
# 或
yarn start
```
